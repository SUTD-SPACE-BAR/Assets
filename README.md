# About
This repository contains the official logos and promotional material of Space Bar, the aerospace initiative of SUTD. Please follow all rules as described by this guide.

# Colour Theme
Space Bar Grey  - #262626
Space Bar Red   - #EF3340
Space Bar White - #F2F2F2

# Design Rules
Official Logo - To be used on either a Space Bar White background, or a plain white background when intended as the subject
Official Logo Inverted - To be used on either a Space Bar Grey background, or a plain black background when intended as the subject
Official Logo, Borderless - To be used on any background darker than #808080 when intended as a footnote / watermark
Official Logo Inverted, Borderless - To be used on any background lighter than or equal to #808080 when intended as a footnote / watermark

# Usage Rules
Use of the logos in any way that is derogatory to club image is strictly prohibited. Logos may only be used publicly with written consent of the club. For more information, contact spacebar@sso.sutd.edu.sg.
